# LrcEdit-Android
安卓版的歌词编辑器
[项目介绍](https://www.licz.site/2018/04/04/%E9%A1%B9%E7%9B%AE%E4%BB%8B%E7%BB%8D/)

已实现功能(v0.1.0)：
* 添加歌词
* 保存为TXT文件(根目录+/lrcEdit下)
* 丑陋的界面
* 较差的用户体验
* 自定义文件名
* 保存为lrc文件格式
* 添加歌词时间

待实现的功能：
* 修改已添加的歌词
* 预览歌词
* 精美的界面
* 良好的用户体验
